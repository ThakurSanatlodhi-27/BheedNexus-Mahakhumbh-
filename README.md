\# BheedNexus - Backend (Mahakumbh Smart Mobility System)



\## Project Overview

\*\*BheedNexus\*\* is the backend system developed using \*\*Spring Boot\*\* to support the \*\*Mahakumbh Smart Mobility System\*\*.  

It manages and monitors crowd movement, police checkpoints, traffic control, and venue operations during large-scale events like the Mahakumbh Mela.  



This backend provides REST APIs that can be consumed by mobile apps or web dashboards to ensure \*\*smart, safe, and efficient crowd management\*\*.



\## Key Features

\- \*\*User Registration \& Authentication\*\* – Secure access for event staff and authorities  

\- \*\*Police \& Venue Management\*\* – Manage personnel and venue assignments  

\- \*\*Traffic Monitoring \& Control\*\* – Real-time traffic data for smart mobility  

\- \*\*Checkpoint Management\*\* – Monitor entry/exit points and crowd flow  

\- \*\*OTP Verification\*\* – Secure validation for users  

\- \*\*Measurement \& Reporting\*\* – Collect and analyze crowd and traffic data



\## Technology Stack

\- \*\*Backend:\*\* Java, Spring Boot  

\- \*\*Database:\*\* H2 / MySQL (configurable)  

\- \*\*Build Tool:\*\* Maven  

\- \*\*Tools:\*\* IntelliJ IDEA, Git, GitHub
